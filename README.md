## ðŸ’» Tech Stack

* **HTML:** For the basic structure and layout.
* **CSS:** For styling, including the game board and player turn indicators.
* **JavaScript :** For all the game logic, including:
    * Handling user clicks (Event Listeners)
    * Updating the game board (DOM Manipulation)
    * Checking for win/draw conditions
    * Resetting the game
    * disabling each button after clicking it once

## âœ¨ Key Features

* Classic 3x3 game board.
* Tracks the current player (X or O).
* Correctly identifies win conditions (horizontal, vertical, and diagonal).
* Announces the winner or a draw.
* A "Reset Game" button to start a new game.
